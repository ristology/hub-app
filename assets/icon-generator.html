Cara pakai:
1. Klik Download 1024×1024 (Master) → file PNG ke-download
2. Replace assets/icon.png di project mobile dengan file ini
3. PNG sudah sudut tajam (no pre-round) → iOS/Android otomatis kasih rounded corner
4. PNG sudah solid background (no alpha) → lolos App Store guideline

Kalau mau adjust warna / shape, edit file ini di text editor — semua angka di <path d="..."> bisa diubah.
